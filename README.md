
# CzechSmallTime

Czech translation of SmallTime version 1.25.1

## Popis
Tento modul poskytuje českou lokalizaci pro SmallTime verze 1.25.1. Obsahuje překlady základních textů aplikace.

## Instalace
Pro instalaci modulu obvyklým upůsobem můžeš použít následující odkaz ve Foundry VTT:

```
https://raw.githubusercontent.com/Lethrendis/CzechTranslationSmallTime/main/module.json
```
